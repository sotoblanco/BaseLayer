# Lesson 3: Attention and RoPE

The heart of the Transformer block is its attention mechanism. Rather than standard Multi-Head Attention, Llama 3 uses a highly efficient Grouped-Query Attention. Additionally, it injects positional information dynamically using Rotary Positional Embeddings (RoPE).

### Your Goal:
Draw the Attention block and the RoPE injection point.

### Instructions:
1. Inside the main dark Transformer block, draw the central box for **Masked grouped-query attention**.
2. To the left, outside the dark box, draw the **RoPE** box.
3. Draw the arrow connecting RoPE into the attention block to show where the positional embeddings are applied!
