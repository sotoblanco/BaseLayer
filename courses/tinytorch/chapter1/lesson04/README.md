## Dot product



We start with the dot product of two vectors. 

The dot product takes two sequence of numbers (vectors) of the same length and returns a single number.

In spreadsheet there are two ways to calculate the dot product of two vectors:

1. Using `SUMPRODUCT` function
2. Using `SUM` and `*` operator

## Matrix multiplication

Matrix multiplication is a series of dot products. To find the value in a specific cell of the resulting matrix, you take the dot product of the corresponding row from the first matrix and the column from the second matrix.

In the spreadsheet you can use `MMULT` function to calculate the matrix multiplication.
