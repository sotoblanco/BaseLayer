# Welcome to the Drawing Tool! 🎨

We've just launched a **new feature**: the Drawing Tool! This tool allows you to interact directly with diagrams and matrices to show your understanding.

### How to Use the Tool
1.  Look at the matrix diagram provided in the exercise.
2.  Use your mouse or touch screen to draw directly on the image.
3.  The **red pencil** is automatically selected for you.

### Your First Task: Matrix Identification
To get started, let's practice identifying values in a matrix.

**Task:**
Find the number **50** in the matrix and draw a **circle** around it using the red pencil.

Once you've marked the number 50, click **Submit Drawing** to have SocratiQ evaluate your work!
